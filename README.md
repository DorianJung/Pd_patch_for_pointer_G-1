# pointer G-1 Pure Data Patch
This patch is inspired by [yann seznec's ys.granular patch](https://github.com/yannseznec/ys.granular), which I used as a foundation to figure out what elements I needed to build my pointer G-1 granular effects unit. The patch has undergone major changes to be usable in a live performance environment, as I needed to integrate it on a headless raspberry pi 5. 

Th pd version I used was 0.56.2 and the following libraries are necessary to run the patch:
- comport
- cyclone
- FluidCorpusManipulation
